💻 開発エンジニア研修

― 3ヶ月で"アプリ"と"システム"の仕組みを学ぶ ―

スノアの研修は、ITが初めての人でも安心です。

親会社が「ITスクール事業」を行っているため、教えるプロが作ったカリキュラムで、基礎からていねいに学べます。

3ヶ月の研修を通して、パソコンの操作からプログラミング、そしてアプリやシステムがどのように作られているのかを理解できます。

【研修期間】3ヶ月（オンラインor対面）

月 学ぶこと 身につく力
1ヶ月目 パソコンの使い方・ITの基本・Webの仕組み パソコン操作やネットの仕組みがわかる
2ヶ月目 プログラミング（Java・Python） コードの意味や動きの仕組みを理解できる
3ヶ月目 チーム開発・システム演習 簡単な課題に取り組みながら実践を体験できる

【サポート体制】

  • 教育経験が豊富な講師が一人ひとりをサポート
  • わからないところはいつでも質問OK
  • 家でも復習できるオンライン教材つき

【学べるスキル】

  • HTML / CSS / JavaScript：Webページの構造や動きを学ぶ
  • Java / Python：アプリやシステムが動く仕組みを理解する
  • SQL / データベース基礎：データを扱う基礎を学ぶ
  • チーム演習（GitHub使用）：開発の流れを体験する

【こんな人におすすめ】

  • アプリやWebサイトがどうやって作られているか知りたい
  • パソコンを使った仕事をしてみたい
  • 将来、エンジニアやIT業界で働いてみたい

未経験者採用

対象となる方

＜本気でITエンジニアを目指したい完全未経験歓迎＞

意欲や積極性を重視した100％ポテンシャル採用です！

基本条件

  • 高卒以上の方
  • 社会人1年以上（バイト含む）
  • 理系・文系は一切問いません
  • IT未経験の方は30歳未満（長期勤続によるキャリア形成のため）

こんな方を歓迎します！（一つでも当てはまればOK）

  • 積極的に学ぶ姿勢がある方
  • 仕事で活かせる専門スキルを身に付けたい方
  • IT業界で活躍したいという思いがある方
  • ゲームやアプリ、Webサービスが好きな方
  • 頑張りや成長を正当に評価されたい方
  • 将来的に高収入を目指したい方
  • IT業界での実務経験がある方（エンジニア・運用・サポートなど職種は不問）
  • IT関係の有資格者（ITパスポート、基本情報、応用情報）

※入社後の研修で基礎から学べるため、現時点での経験や資格は一切不問です！
